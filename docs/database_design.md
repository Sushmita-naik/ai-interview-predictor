# AI Interview Predictor Database Design

## Database Name

```text
interview_predictor.db
```

---

# Users Table

Stores user account information.

| Column   | Type    | Description   |
| -------- | ------- | ------------- |
| id       | INTEGER | Primary Key   |
| name     | TEXT    | User Name     |
| email    | TEXT    | Unique Email  |
| password | TEXT    | User Password |

SQL:

```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    email TEXT UNIQUE NOT NULL,
    password TEXT NOT NULL
);
```

---

# Resumes Table

Stores uploaded resume details.

| Column      | Type      | Description      |
| ----------- | --------- | ---------------- |
| id          | INTEGER   | Primary Key      |
| user_id     | INTEGER   | User ID          |
| resume_name | TEXT      | Resume File Name |
| resume_path | TEXT      | Resume File Path |
| uploaded_at | TIMESTAMP | Upload Time      |

SQL:

```sql
CREATE TABLE resumes (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    resume_name TEXT,
    resume_path TEXT,
    uploaded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY(user_id) REFERENCES users(id)
);
```

---

# Interview Results Table

Stores interview evaluation scores.

| Column              | Type    | Description         |
| ------------------- | ------- | ------------------- |
| id                  | INTEGER | Primary Key         |
| user_id             | INTEGER | User ID             |
| technical_score     | REAL    | Technical Score     |
| communication_score | REAL    | Communication Score |
| project_score       | REAL    | Project Score       |
| confidence_score    | REAL    | Confidence Score    |
| total_score         | REAL    | Final Score         |

SQL:

```sql
CREATE TABLE interview_results (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    technical_score REAL,
    communication_score REAL,
    project_score REAL,
    confidence_score REAL,
    total_score REAL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY(user_id) REFERENCES users(id)
);
```

---

# Placement Predictions Table

Stores placement prediction results.

| Column                | Type    | Description      |
| --------------------- | ------- | ---------------- |
| id                    | INTEGER | Primary Key      |
| user_id               | INTEGER | User ID          |
| placement_probability | REAL    | Placement Chance |
| salary_prediction     | REAL    | Predicted Salary |

SQL:

```sql
CREATE TABLE placement_predictions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    placement_probability REAL,
    salary_prediction REAL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY(user_id) REFERENCES users(id)
);
```

---

# Interview Questions Table

Stores generated interview questions and answers.

| Column   | Type    | Description         |
| -------- | ------- | ------------------- |
| id       | INTEGER | Primary Key         |
| user_id  | INTEGER | User ID             |
| question | TEXT    | Generated Question  |
| answer   | TEXT    | User Answer         |
| score    | REAL    | AI Evaluation Score |

SQL:

```sql
CREATE TABLE interview_questions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    question TEXT,
    answer TEXT,
    score REAL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY(user_id) REFERENCES users(id)
);
```
